# Auto Claw
The ultimate autonomous orchestrator. Can perform system-level tasks, write code, and organize files just from a prompt.

## When to use
- "Organize my desktop into folders based on file type."
- "Create a Python script that monitors my CPU temperature."
- "Find all large files in my Downloads and move them to an archive."
- "Write and run a script to scrape X and save to a CSV."

## Capabilities
- Autonomous Python execution
- Shell command access
- File system manipulation
- Multi-step logic processing
